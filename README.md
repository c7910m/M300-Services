## Markdown

https://code.visualstudio.com/docs/languages/markdown

Markdown wird zum darstellen einer Datei gebraucht in einem "Humanen Stil".


<p> \# H1 Übertitel 1 </p>
<p> \## H2 Übertitel 2 </p>
<p> \### H3 Übertitel 3 </p>
<p> \#### H4 Übertitel 4 </p>
<p> \##### H5 Übertitel 5 </p>
<p> \###### H6 Übertitel 6 </p>
